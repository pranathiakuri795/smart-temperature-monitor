# 🌡️ Smart Temperature Monitor

## 📌 Project Description
The Smart Temperature Monitor is an embedded system project that measures temperature and provides an alert when the temperature crosses a predefined threshold value. It helps in real-time monitoring of temperature for safety and control applications.

---

## 🎯 Objective
The main objective of this project is to monitor temperature continuously and give an alert signal when it goes above a safe limit.

---

## ⚙️ Features
- Real-time temperature sensing  
- Threshold-based alert system  
- Simple and low-cost embedded design  
- Useful for home and industrial safety applications  

---

## 🧰 Components Used
- Temperature Sensor  
- Microcontroller  
- Buzzer / LED (for alert)  
- Power Supply  

---

## 📊 Working Principle
The temperature sensor continuously reads the surrounding temperature. The microcontroller compares the value with a set threshold. If the temperature exceeds the limit, the alert system (buzzer or LED) is activated.

---

## 🚀 Applications
- Room temperature monitoring  
- Industrial safety systems  
- Smart home systems  
- Cooling system control
